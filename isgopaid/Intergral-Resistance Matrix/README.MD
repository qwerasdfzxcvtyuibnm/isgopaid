There are integral-resistance matrices  for the 12-round PRESENT and the 11-round PRESENT.



Type 'sage determine_rank.sage 12' to solve the rank of 12-round integral-resistance matrix.

Type 'sage determine_rank.sage 11' to solve the rank of 11-round integral-resistance matrix.


The version of sagemath we used is  "SageMath version 9.7, Release Date: 2022-09-19".
